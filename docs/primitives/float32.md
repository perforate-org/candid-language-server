IEEE-754 single-precision floating point numbers.

---

## Textual syntax

Same as `int`, plus floating point literals:

```candid
1245.678
+1245.678
34e-10
0xDEAD.BEEFP-10
```

---

## Corresponding types/values

### Motoko types

_Not currently supported in Motoko._

### Rust types

```rust
f32
```

### JavaScript values

```javascript
Number // coerced to double precision
```
