Signed 32-bit integers (`-2_147_483_648..2_147_483_647`).

---

## Textual syntax

Same as `int`; annotate to disambiguate.

```candid
-1 : int32
```

---

## Corresponding types/values

### Motoko types

```motoko
Int32
```

### Rust types

```rust
i32
```

### JavaScript values

```javascript
Number
```
